# Reminder Program

## About
This is a small program that works as a reminder. You can make simple, meeting, or daily reminders. Program saves all reminders in a JSON file.

## Features
* Simple reminder
* Meeting reminder with participants
* Daily reminder (repeat)
* Show all reminders
* Find reminder by ID
* Run all reminders
* Group by type

## How to use
Run:
python main.py

Follow menu:
1. Make new reminder
2. Remove reminder
3. Show all
4. Find by ID
5. Run all
6. Group type
7. Exit
